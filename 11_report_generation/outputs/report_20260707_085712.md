# 铁矿石45港库存预测模型结果与走势分析报告

## 核心结论

铁矿石库存预测模型（LightGBM）在测试集上实现了 **84.91% 的方向准确率（MDA）** 和 **2.87% 的平均绝对百分比误差（MAPE）**，整体方向预测能力较好。但从最近 12 周序列来看，预测值**系统性低于实际值约 1,031–1,657 万吨**，模型未能充分捕捉库存高位波动与"先升后降"倒 U 形走势的幅度。实际库存于 2026-03-13 触及峰值 17,187.52 万吨后逐步回落至 5 月初的 16,541.80 万吨，预测值则维持在 15,410–15,573 万吨区间内窄幅波动。训练集与测试集 MDA 差距 14.65 个百分点，存在一定过拟合，绝对值预测精度有限，使用时需注意低估偏差。

## 二、模型评估与表现分析

### 2.1 核心评估指标

| 指标 | 训练集 | 测试集 |
|------|--------|--------|
| MDA（方向准确率） | 99.56% | **84.91%** |
| MAPE（平均绝对百分比误差） | 0.026% | **2.87%** |
| MAE（平均绝对误差，万吨） | 3.35 | **477.38** |

- **方向准确率**：测试集 MDA 为 84.91%，即约 85% 的周度方向变动被正确预测，方向预测能力较好。
- **MAPE**：测试集平均绝对百分比误差为 2.87%，预测值平均偏离实际值约 2.87%。
- **MAE**：测试集平均绝对误差为 477.38 万吨，相对于库存量级（约 1.37–1.72 亿吨）属于中等水平。

### 2.2 最优超参数配置

| 超参数 | 值 |
|--------|-----|
| num_leaves | 187 |
| max_depth | 4 |
| learning_rate | 0.0201 |
| n_estimators | 1,664 |
| min_child_samples | 30 |
| subsample | 0.6367 |
| colsample_bytree | 0.6026 |
| reg_alpha | 2.44e-07 |
| reg_lambda | 9.37e-04 |
| random_state | 42 |

模型配置以**低学习率（0.02）、中等树深（4）、高叶子数（187）**为特征，采用 200 次 Optuna 试验搜索最优超参组合。

### 2.3 过拟合分析

模型存在**一定程度的过拟合**，主要依据：

1. **MDA 差距 14.65 个百分点**：训练集方向准确率近乎完美（99.56%），测试集降至 84.91%。
2. **MAE 训练/测试比悬殊**：训练集 MAE 仅 3.35 万吨，测试集高达 477.38 万吨，差距超过 140 倍。
3. **特征数远超样本数**：3,184 个特征对应 470 行训练数据，特征维度远高于样本量。
4. **正则化偏弱**：reg_alpha 为 2.44e-07（极小），reg_lambda 为 9.37e-04，L1/L2 正则化力度较低。

### 3.2 库存走势方向分析

基于最近 12 周数据观察：

- **实际库存走势**：呈现**先升后降的倒 U 形**。2 月中旬至 3 月中旬持续上升（16,946 → 17,188，峰值在 3 月 13 日），3 月下旬起逐步回落至 5 月初的 16,542 万吨，累计下降约 646 万吨。
- **预测库存走势**：整体**较为平坦**，预测值在 15,411 ~ 15,573 万吨之间小幅波动，未能充分捕捉实际库存的上升与下降幅度。
- **最新一期预测**：2026-05-01 预测值为 **15,510.69 万吨**，实际值为 16,541.80 万吨，绝对误差 1,031.12 万吨。

### 3.3 预测偏差特征

- **系统性低估**：预测值系统性低于实际值约 1,000–1,660 万吨，存在明显的低估倾向。
- **误差峰值在库存高点**：绝对误差在库存峰值附近（3 月 13 日）最大，达 1,657.49 万吨。
- **误差收敛趋势**：绝对误差从 3 月中旬的峰值 1,657 万吨逐步收窄至 5 月初的 1,031 万吨，表明模型在库存下行阶段的预测偏差有所改善。
- **近期误差高于整体均值**：最近 12 周的绝对误差区间（1,031–1,657 万吨）显著高于测试集整体 MAE（477.38 万吨），说明这一时段对应库存高位波动期，预测难度较大。

## 总结

铁矿石库存预测模型在**方向预测层面表现可靠**（测试集 MDA 84.91%），能够为库存走势方向提供有价值的参考。但从最近 12 周序列来看，模型存在**系统性低估偏差**，预测值整体低于实际值约 1,000–1,660 万吨，未能充分捕捉库存高位波动与倒 U 形走势的幅度。实际库存已于 2026-03-13 触及 17,187.52 万吨的峰值后持续回落，最新一期（2026-05-01）实际值为 16,541.80 万吨，预测值为 15,510.69 万吨。

模型存在一定过拟合（训练/测试 MDA 差距 14.65 个百分点），绝对值预测精度有限。在使用该模型输出时，建议重点关注其**方向预测信号**，对绝对数值预测保持审慎，并结合其他市场信号综合判断。
