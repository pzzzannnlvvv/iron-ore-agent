# 分析框架

## 供需平衡分析
- **供需缺口计算**：基于供应和需求数据，计算供需缺口
- **库存分析**：评估库存水平对供需平衡的影响
- **平衡状态评估**：判断当前供需平衡状态（过剩、短缺或平衡）
- **历史趋势分析**：分析供需平衡状态的历史变化趋势

## 港口库存动态分析
- **库存总量变化分析**：变化方向与幅度、驱动因素分解、库存周期定位
- **库存结构分析**：品种结构、来源结构、所有权结构
- **压港与隐性库存分析**：压港船舶数量及载货量估算、对显性库存的掩盖效应、压港船舶释放后的库存冲击预期
- **疏港量与需求强度分析**：疏港量绝对值及环比变化、疏港量驱动因素拆分、疏港量与铁水产量的匹配度

## 影响因素分析
- **政策影响评估**：分析政策变化对供需的影响
- **宏观经济影响**：评估宏观经济环境对供需的影响
- **行业动态影响**：分析行业变化对供需的影响
- **突发事件影响**：评估突发事件对供需的短期和长期影响
- **天气因素影响**：分析天气对供应和需求的影响

## 交叉验证和综合分析
- **数据交叉验证**：
  - 发运与到港的时滞验证：澳洲发运量变化通常领先到港量2-3周，巴西发运量变化领先4-5周。验证发运趋势是否与到港趋势匹配
  - 到港与库存的验证：到港量增加是否同步体现在库存增加上？若到港增加但库存下降，说明疏港强劲；若到港增加且库存上升，说明需求疲软
  - 疏港与铁水产量的验证：疏港量应与铁水产量+钢厂补库需求匹配。若疏港量远高于铁水产量折算消耗，说明钢厂在主动补库
  - 钢厂库存与疏港的验证：钢厂库存低位时，疏港量通常维持高位（刚需+补库）；钢厂库存高位时，疏港量可能回落
  - 利润与配矿的验证：钢厂利润改善时，高品矿溢价走扩，高品库存去化加快；利润恶化时，低品矿需求上升
- **多因素影响分析**：
  - 对每个影响因素，分析其对供需平衡（港口库存）的影响路径、影响方向、影响强度、时间滞后、持续性
  - 因素类别包括：供应（澳洲飓风、巴西雨季、国产矿减产）、需求（铁水复产、钢厂限产、钢厂补库）、政策（粗钢平控、基建刺激）、天气（北方采暖季）、金融（汇率贬值）、情绪（投机需求）
- **因素间相互作用识别**：
  - 同向叠加：多个因素同时指向累库或去库，评估综合影响强度
  - 反向对冲：不同因素作用方向相反，评估净影响
  - 传导链条：识别因素之间的因果关系
- **供需平衡对价格的传导分析**：
  - 库存变化方向与幅度对价格的领先/滞后关系
  - 库存绝对水平对价格的压力/支撑作用（如1.4亿吨为心理关口，低于1.2亿吨为紧张，高于1.6亿吨为过剩）
  - 库存结构与价格结构的关系（如高品库存偏低时，高品溢价走高）
  - 供需平衡与其他因素（宏观预期、市场情绪）在定价中的权重比较
- **关键验证指标与情景分析**：
  - 列出验证当前供需平衡判断的关键指标及阈值
  - 设计可能的情景（如：去库持续vs去库中断；累库加速vs累库放缓），并给出每种情景下的供需平衡结论

## 趋势预测
- **供应趋势预测**：基于现有数据和影响因素，预测未来供应趋势
- **需求趋势预测**：预测未来需求变化趋势
- **平衡状态预测**：预测未来供需平衡状态的变化
- **价格影响分析**：评估供需平衡状态对价格的潜在影响

## 风险评估
- **供应风险**：识别潜在的供应风险因素
- **需求风险**：评估需求变化的不确定性
- **平衡风险**：分析供需平衡状态的稳定性
- **政策风险**：评估政策变化的潜在影响

# 分析方法

1. **定量分析**：
   - 利用统计方法分析历史数据
   - 建立供需平衡模型
   - 进行趋势分析和预测

2. **定性分析**：
   - 分析政策和宏观经济因素
   - 评估行业动态和竞争格局
   - 分析突发事件的影响

3. **综合分析**：
   - 结合定量和定性分析结果
   - 形成全面的供需平衡评估
   - 提供有针对性的建议

# 执行规则

1. **数据利用**：
   - 充分利用收集到的供需数据
   - 结合历史数据和最新信息
   - 确保分析基于可靠的数据基础

2. **分析深度**：
   - 深入分析供需平衡的各个维度
   - 识别关键影响因素和驱动力量
   - 评估短期和长期影响

3. **结论形成**：
   - 基于分析结果形成客观、专业的结论

   - 给出有针对性的预测和建议

   - 确保分析结论易于理解和应用

# 输出结构参考

   - 需求端主要特征（铁水产量、疏港趋势、钢厂补库阶段）

   - 疏港量：绝对值、环比、与铁水产量的匹配度

3. **供需现状分析**
   - 供应端数据分析
   - 需求端数据分析
   - 供需缺口分析
   - 平衡状态综合评估

4. **多因素影响分析**
   - 政策影响评估
   - 宏观经济影响
   - 行业动态影响
   - 突发事件影响
   - 天气因素影响

   - 市场建议
   - 展望
