# 分析框架

## 1. 价格表现与结构

- 复盘普氏61%指数、大商所铁矿石主力合约、青岛港主流品牌现货的阶段性表现。
- 分析期现基差、期限结构、品牌价差、块矿/球团溢价变化；若球团价格、基差、期限结构或持仓量未覆盖，必须在“数据缺口与结论可靠性”中点名。
- 对周报结构化指标，若分析周期内日期缺失但观察窗内存在最近可得数据，必须标注“最近可得日期”，并避免把两周观察窗的全部波动等同于本周涨跌。
- 识别价格变化主要由供应、需求、宏观、资金或事件中的哪类因素驱动。

## 2. 供需基本面

- 供应端：四大矿山发运、国内到港、非主流矿扰动、国内矿山开工、港口库存总量与结构。
- 供应端库存优先识别47港口库存；若只有其他港口口径，必须说明口径差异。需求端：247家钢厂日均铁水产量、钢厂利润、高炉/烧结开工、进口矿库存天数、终端需求传导；若铁水产量来自舆情转述而非结构化数据，必须降低结论强度。
- 建立“发运/到港-库存-铁水-利润-价格”的逻辑链条。

## 3. 宏观政策与事件

- 国内货币财政政策、地产预期、粗钢压控、环保限产等对黑色产业链的影响。
- 海外美元指数、利率预期、全球风险偏好对商品定价的影响。
- 关键事件按影响方向和持续性排序，区分短期扰动与趋势性因素。
- 周报关键事件只使用分析周期内发生或发布的信息；不要纳入结构化指标观察窗中更早一周的政策和舆情。

## 4. 展望与策略

- 基于已收集证据形成未来1-4周价格方向判断。
- 若已有足够价格和驱动数据，可以给出目标区间；若数据不足，只给出方向性判断和触发条件，不虚构上下沿。
- 策略建议应覆盖采购节奏、期现/跨期/品种套利机会、风险对冲和止损触发条件。
