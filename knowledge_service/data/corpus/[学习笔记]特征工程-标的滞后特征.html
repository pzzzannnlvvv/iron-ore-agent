2.2 滞后特征（Lag Features）

短期、中期、长期滞后特征
用过去的库存值作为预测现在的特征
概念解释

什么是滞后特征？

  • 比如今天是 t，昨天是 t-1，前天是 t-2
  • 用昨天（t-1）的库存作为特征来预测今天（t）的库存
  • 这就是：target_lag_1

为什么要分短期、中期、长期？

  • 短期：最近几天的库存变化，反应短期趋势
  • 中期：半个月到一个月，反应中期趋势
  • 长期：一季度到一年，反应季节性、周期性规律
代码实现
short_lags = [1, 2, 3, 5, 7]
for lag in short_lags:
    arr = np.full(n, np.nan, dtype=np.float32)
    arr[lag:] = target_data[:-lag]
    features[fid] = arr
逐行解释
short_lags = [1, 2, 3, 5, 7]
小白理解：定义短期滞后天数：1天、2天、3天、5天、7天
np.full(n, np.nan)
小白理解：先创建一个全是NaN的数组
arr[lag:] = target_data[:-lag]
小白理解：核心！把「过去的数」放到「现在的位置」

2.3 差分特征（Difference Features）

target_diff_1, target_diff_7
库存的变化量：今日库存 - 昨日库存
概念解释

什么是差分？

  • 差分 = 现在的值 - 过去的值
  • 差分能看出来「库存是在涨还是跌」

差分 > 0 → 库存增加

差分 < 0 → 库存减少

差分 ≈ 0 → 库存稳定

代码实现
diff_arr = np.full(n, np.nan, dtype=np.float32)
diff_arr[lag:] = target_data[lag:] - target_data[:-lag]
逐行解释
target_data[lag:] - target_data[:-lag]
小白理解：用「现在的数据」减去「过去的数据」，得到变化量
举个例子
日期库存diff_1说明
Day 1100NaN
Day 21055涨了5
Day 3102-3跌了3
Day 41086涨了6

2.4 变化率特征（Percent Change Features）

target_pct_1, target_pct_7
库存的变化率，用百分比表示
概念解释

什么是变化率？

  • 变化率 = (现在的值 - 过去的值) / 过去的值
  • 好处是可以对比不同基数的变化
举个例子：为什么要用变化率？
库存diff_1pct_1
A仓库100 → 10555%
B仓库1000 → 1030303%

虽然B仓库涨了30（绝对值更多），但其实A仓库涨幅更大（5% > 3%）

代码实现
safe_denom = np.where(np.abs(target_data[:-lag]) > 1e-10,
                      target_data[:-lag], np.nan)
ratio = (target_data[lag:] - target_data[:-lag]) / safe_denom
逐行解释
np.where(np.abs(target_data[:-lag]) > 1e-10, ...)
小白理解：避免除以0！如果过去的值接近0，就设为NaN
ratio = (现在 - 过去) / 过去
小白理解：计算变化率
